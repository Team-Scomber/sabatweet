SabaTweet is a new Twitter client

SabaTweet はツイッターヘビーユーザのための新しいツイッタークライアントです。もちろん初心者()にも使いやすいよう配慮されています ;)

このアプリケーションはバックエンドにRDBMSを持ち、確実に読み込んだ(あるいはあなたがポストした)ツイートを保護します。

downloads

右のボタンからダウンロードして下さい。(.zipと.tar.gzの内容はどちらも同じです)

インストール

お好みの場所にアーカイブを展開して下さい。展開後、同梱されているSSCERuntime-JPN.exeを起動し Microsoft SQL Server Compact 3.5 SP2 をインストールして下さい。

その後、sabatweet.exe をダブルクリックするとサバツイートが起動します。

特長

  • Twitter表示要求(Display Requirements)に完全準拠
  • 画像のインプレースプレビュー

system requirements

以下のソフトウェア実行環境を要求します。開発者は Windows XP と Windows7 上で開発を行っています。
  • Windows XP 以上
  • .NET Framework 4.0 以上
  • Microsoft SQL Server Compact 3.5 SP2(起動しない場合インストールして下さい)
